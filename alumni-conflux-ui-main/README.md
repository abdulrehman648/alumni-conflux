#  Alumni Conflux UI

A modern mobile app UI built using React Native (Expo) for connecting students with alumni mentors.

---

##  Features

###  Student
- Browse mentors
- Book sessions
- Cancel sessions
- Give ratings & reviews
- Profile management

###  Alumni
- Manage availability
- Accept / Reject session requests
- View session history

###  Admin
- Manage users
- Manage mentors
- Manage events

---

##  Tech Stack

- React Native (Expo)
- Expo Router
- Context API
- React Hooks
